# to_top

A super-light "Back to top" jQuery function.

## How to use

```js
$('#my-awesome-back-to-top').to_top();
```

## $.fn.to_top

* `limit`: limit value your "Back to top" will display, default = 300
* `speed`: time to go to top, default = 400
